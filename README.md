# CBC Progress Tracker 2025

A mobile-first React application designed for parents to track their children's progress in the Competency-Based Curriculum (CBC).

## Features
- **Learner Profiles:** View individual progress for students.
- **Milestone Tracking:** Visualize performance across different subjects (Literacy, Numeracy, etc.).
- **CBC Status Indicators:** Clearly mark outcomes as Exceeding, Meeting, or Approaching expectations.
- **Responsive Design:** Optimized for mobile viewing on smartphones.

## Setup Instructions
1. Install dependencies: `npm install`
2. Start development server: `npm start`
3. Built with React, TypeScript, Lucide-React, and Tailwind CSS.
